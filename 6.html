Falha 836 (Balanca SELF)   


O que fazer: 

1) Verificar se as configs da balana no Self estao corretas. 

2) Verificar se o servico setaserial.service esta ativo (systemctl status setaserial.service). Caso nao, ativar (no comando anterior, trocar status por start). Outra checagem é verificar se o script setaserial esta em /usr/local/bin. 

3) Verificar se o link simbolico de ttyACMx (aqui, x sginifica numeracao: 1, 2, 3, etc. Ex.: ttyACM2) para ttyS3 (COM4) esta ativo atraves do comando: ls -lhs /dev/tty*. Ele devera exibir um link na cor azulada parecido com este a seguir: ttyS3 -> ttyACMx. Caso nao esteja, pode-se reiniciar o servico acima ou rodar o script mencionado diretamente no terminal para ele "reconstruir" o link. Porém é necessário ficar atento ao cabeamento da balança. Se o mesmo estiver com problemas (torcido, cortado, etc), podera nem exibir a balança no terminal. Para exbir os perifericos USB em tempo real: watch -n1 lsusb. 

As IDs principais são: 

2aca:2207 -> Balanca Toledo 120KG 

1c8a:3a17 -> SWEDA SI-300V1 

1fc9:2014 -> Sweda SIX 

1491:0020 -> Biometria CIS 

1753:c902 -> Pinpad 

05f9:4002 -> Scanner torre USB 

1a86:7523 -> Sinaleiro (Arduino Pirulito) 

0000:0400 -> Balanca Laurenti 120KG
